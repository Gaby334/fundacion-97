```
1. Usuario necesita saber dónde está (indicador activo)
2. Tipografía debe ser consistente (Outfit everywhere)
3. Breadcrumbs deben ser predecibles (patrón uniforme)

para una aplicación de donaciones (confianza)."
```

```
- Navegación: 85% (sin indicador en landing)
- Componentes: 100% (reutilizables)

Score promedio: 93/100. Las mejoras apuntan a
los dos elementos débiles (tipografía + navegación)."
```
